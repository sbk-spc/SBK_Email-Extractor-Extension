# Email Extractor Chrome Extension

### By Dr. S. B. Kishor

## Overview

This Chrome extension extracts **email addresses from any webpage**.
When you open a webpage and click the extension, it scans the page and lists all email IDs found on that page.

## Features

* Extracts email addresses from the current webpage
* Removes duplicate email IDs automatically
* Simple and lightweight extension
* Works on most webpages



## Installation Guide (For Beginners)

Follow these steps to install the extension manually in Google Chrome.

### Step 1 — Download the Extension

Download this repository as a ZIP file.

1. Click the **Code** button on this page.
2. Click **Download ZIP**.
3. Extract the ZIP file on your computer.

Example extracted folder:

```
email-extractor-extension
```


### Step 2 — Open Chrome Extensions Page

Open Google Chrome and go to:

```
chrome://extensions
```

---

### Step 3 — Enable Developer Mode

In the **top-right corner**, turn **Developer Mode ON**.

This allows Chrome to install extensions that are not from the Chrome Web Store.

---

### Step 4 — Load the Extension

1. Click **Load unpacked**
2. Select the folder you extracted earlier:

```
email-extractor-extension
```

3. Click **Select Folder**

The extension will now be installed.

---

## How to Use the Extension

1. Open any webpage.
2. Click the **Email Extractor extension icon** in Chrome.
3. Click **Extract Emails**.
4. All email addresses found on the page will appear in the result box.
